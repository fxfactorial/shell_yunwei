Smalltalk
=========

    这是 Dolphin Smalltalk 2.1 的完全安装版本，包括说明文档。


    要使用这个教程，须运行以下安装程序:

    DolphinCDSetup.exe

   然后按照提示操作。

    安装过程中，可以看到说明文档，名称为"Education Centre" 。本教程由以
下几部分组成：

  初学指南:
  BeginnersGuide/BeginnersGuide.htm

  几个练习:
  Tutorials/Tutorials.htm

  还有其他一些参考资料。


  感谢 Andy Bower为我们提供了这个教程，可以通过以下网址与他联系：

  http://www.object-arts.com/
