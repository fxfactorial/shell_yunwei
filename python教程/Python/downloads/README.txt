IDLE05   ZIP       108,959  03-06-00  9:37p idle05.zip
IDLEUS~1 ZIP       106,546  03-06-00  9:38p idleUse05b.zip
----------------------------------------------------------
这是IDLE的升级版本，在Python 1.5.2中发布. 它对原版本进行了一些改进，但改进并没有实质性的用途。

使用改进版本时，首先删除或重命名已经存在的idle文件夹，然后把zip文件解压到Python 1.5.2安装目录的
Tools文件夹，再把解压文件名称更改为 idle;或者，更改Start|Programs 菜单中的程序名，使之与解压文件
一致。


WIN32A~1 EXE     3,023,760  08-21-99 12:32p win32all-132.exe
------------------------------------------------------------
这是Mark Hammond为Python提供的Windows tools工具集。

通过它可以访问MFC API and Windows Script Host，以及称做Pythonwin的IDE 和IDLE 替代产品。Pythonwin
提供有一个非常好的调试模式，所以你可以把Pythonwin设置成默认的Python文件编辑器。


LIFE-P~1 GZ         35,758  02-15-99  5:57p life-preserver-html_tar.gz
----------------------------------------------------------------------
这是Matt Conway公司提供的Tkinter实时观测器，虽然有些过时，但仍然是一个快速熟悉Tkinter的产品，它提
供了一些与配套书中代码相同的示例。Python 网页上包有更多的Tkinter信息。

SPECPY~1 TGZ        14,742  03-08-99 12:11p SpecPython0_2.tgz
-------------------------------------------------------------
SpecTcl 是为Tcl, Perl, Java 和 HTML提供的图形用户界面构建程序，虽然它不是最复杂的工具，但却提供有
非常详尽的说明文档。如果添加SpecPython组件，则可以通过它来生成Tkinter代码。 你最好能研究一下网格
布局管理程序（grid layout manager），因为 SpecTcl使用的是这个管理程序，而不是书中介绍的打包程序（
packer）。
